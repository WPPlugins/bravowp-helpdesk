== Description ==
A freeware solid Support Tickets manager for your WordPress portal. With a complete front-end content included, you can allow your 
vistors to open Support Tickets directly from your website. Completely build on Twitter Bootstrap it offers a clean and easy-to-
use Administration interface to fully manage all the Helpdesk elements. Check the description below to see what is included in the 
Freeware version, and what can be added with our add-ons shop.

Features:

* Responsive, completely built with Bootstrap framework.
* Wrapped layout, will not mess up your theme.
* 100% AJAX based, for super fast user experience.
* Custom MySql tables, will not interfere with your WP content.
* Administration back-end Support Tickets Management.
* Ticket Status management.
* Ticket Category management.
* Super fast Tickets search with client-side tabled data.
* Dashboard with Status counters, last 10 days Statistics chart, and more.
* Create Ticket on customer's behalf.
* Administrators and Operators can add Messages to Ticket.
* System Event log for debugging purposes.
* Full no restriction front-end.
* Allow your customers to create and review their Tickets.
* Customers can add messages to their existing Tickets.
* CAPTCHA control security (setting).
* Short-code to render front-end content on your WP page.
* Confirmation page after Ticket creation.
* Tickets can be created from non-registered visitors (setting).
* Fully localizable.
* Made in Italy, where also Pizza was created. Enough said.

== Installation ==
* Login as Administrator of your portal;
* Navigate to Plugins -> Add New;
* Click on Upload;
* Select and Upload the .zip file you downloaded from this website;
* Activate the new plugin.
* Short code for Front End: [bravowp-helpdesk-frontend]
